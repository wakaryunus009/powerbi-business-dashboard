# Business Intelligence Dashboard for Retail Operations

## Tools Used
- Microsoft Excel (Power Query, Power Pivot)
- Power BI Desktop
- DAX

## Overview
This project demonstrates how raw retail sales data can be transformed into an interactive Power BI dashboard
to track KPIs such as revenue, profit, profit margin, and sales trends.
